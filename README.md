## Setup

1. Clone the repo:  
   ```bash
   git clone https://github.com/jeffreyhasan10/jeffrey.dev.git
   ```
2. Install dependencies:  
   ```bash
   npm install
   ```
3. Run locally:  
   ```bash
   npm run dev
   ```
